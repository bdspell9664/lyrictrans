Lyrics Translator Proxy

这是一个用于歌词翻译工具的GitHub Pages代理服务器，用于解决跨域问题。

API Endpoint

POST /translate

请求参数

q: 要翻译的文本
from: 源语言（默认：auto）
to: 目标语言
appid: 百度翻译APP ID
secretKey: 百度翻译密钥
salt: 随机数
sign: 签名

使用示例

fetch('https://your-username.github.io/lyrics-translator-proxy/translate', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/x-www-form-urlencoded'
    },
    body: new URLSearchParams({
        q: 'Hello World',
        from: 'en',
        to: 'zh',
        appid: 'your-app-id',
        secretKey: 'your-secret-key',
        salt: '123456',
        sign: 'your-sign'
    })
})
.then(response => response.json())
.then(data => console.log(data));

部署说明

  1. 将此仓库fork到您的GitHub账户
  2. 进入仓库设置，开启GitHub Pages
  3. 在歌词翻译工具的配置中，将代理URL设置为您的GitHub Pages URL

注意事项

  • 此代理仅用于歌词翻译工具
  • 请勿滥用，遵守百度翻译API的使用条款
  • 由于GitHub Pages的限制，代理请求可能会有延迟
